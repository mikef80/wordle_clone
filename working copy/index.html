About Colourdle!


One new Colourdle! will be available each day.


Colourdle! has been heavily inspired by Wordle!, which was created by Josh Wardle and is now run by the NYT


It was inspired by a friend (Sam) who suggested this as an idea and has been created by Mike, both for fun and to practice his coding skills in HTML, CSS and JavaScript.


It is available to play for free - but if you'd like to support it, please feel free to buy me a coffee!


Welcome to Colourdle!


The object of the game is to correctly guess the RGB value of the target colour swatch within x tries.


Each guess must be a valid colour.


The first three columns are for the R(ed) value, the next three are for the G(reen) value and the last three are for the B(lue) value. All colour codes must be three digits long, i.e. you must use leading zeros (for example, 053, not 53). All colour codes have a maximum value of 255.


At the end of the line, press or click enter to submit your answer.


- If a square is green then the number in that spot was correct.


- If a square is yellow then the number is in the colour code somewhere, but is in the wrong spot.


- If the square is grey then the number is not included in the colour code.


The “Last Guess” square will show you which colour you actually entered and how it compares to the targe colour.
